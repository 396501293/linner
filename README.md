# Linner

HTML5 Application Assembler

## Installation

    $ gem install linner

## Usage

Skeleton:

    $ linner new webapp && cd webapp

Watch:

    $ linner watch

Server:

    $ ./bin/server # or server if put "./bin" in your PATH

Build:

    $ linner build

Clean:

    $ linner clean
